🧩 ШАГ 2: ПСЕВДОКОД (шаблон)

Функция (вход) -> (выход):
1. Если вход = nil → подставить значение по умолчанию (??)
2. Проверить, что данные не пустые (guard)
3. Если пусто → вернуть ошибку
4. Преобразовать данные (Int, String, и т.д.)
5. Если преобразование не удалось → вернуть ошибку
6. Вернуть результат

🧩 ШАГ 3: БЛОК-СХЕМА (шаблон)

[ Начало ]
    ↓
[ Подставить значения по умолчанию (??) ]
    ↓
[ Данные пустые? ] — ДА → [ Ошибка ] → [ Конец ]
    ↓ НЕТ
[ Преобразовать данные ]
    ↓
[ Преобразование удалось? ] — НЕТ → [ Ошибка ] → [ Конец ]
    ↓ ДА
[ Вернуть результат ]
    ↓
[ Конец ]

🧩 ШАГ 4: ПРИМЕР (задача с опционалами)

// Псевдокод
func formatUser(name: String?, age: String?) -> String {
    // 1. Подставить значения по умолчанию
    let safeName = name ?? ""
    let safeAge = age ?? ""
    
    // 2. Проверить, что имя не пустое
    guard !safeName.isEmpty else { return "Данные неполные" }
    
    // 3. Проверить, что возраст не пустой
    guard !safeAge.isEmpty else { return "Данные неполные" }
    
    // 4. Преобразовать возраст в число
    guard let ageNum = Int(safeAge) else { return "Данные неполные" }
    
    // 5. Вернуть результат
    return "Имя: \(safeName), возраст: \(ageNum)"
}
