☕️💳 КОММЕРЧЕСКИЙ КЕЙС: ЛЕНТА БАНКОВСКИХ КАРТ (USER CARDS)

📌 Бизнес-требование:
    У каждого клиента банка может быть разное количество карт (у кого-то одна зарплатная, у кого-то три — для миль, кэшбека и валютная). 
    В базе данных они хранятся в виде вложенных списков. 
    Экран приложения запрашивает карту по двум номерам: «Дай мне cardIndex для пользователя userIndex». 
    Код должен работать гибко и не падать, если у конкретного пользователя карт меньше, чем у соседа.
    
    📝 Условие задачи:
        Создайте структуру BankDatabase. Внутри неё находится двумерный массив строк, где каждый внутренний массив — это кошелек конкретного клиента:
        
        struct BankDatabase {
    var userCards: [[String]] = [
        ["Мир", "Visa"],      // Клиент 0 (Алексей) — 2 карты
        ["Mastercard"],       // Клиент 1 (Мария) — 1 карта
        ["Мир", "UnionPay"]   // Клиент 2 (Иван) — 2 карты
    ]
    
    // НАПИШИТЕ ВАШ КАСТОМНЫЙ САБСКРИПТ ТУТ
}

🛠️ ТЗ:
1. Сабскрипт должен принимать два параметра: userIndex: Int и cardIndex: Int.
2. На выходе он должен отдавать строку -> String.
3. Блок GET (Чтение):
    План А (Безопасно): 
            Сначала проверьте, что userIndex вообще существует в массиве userCards (индекс от 0 до 2).
            Затем проверьте, что cardIndex существует внутри массива карт именно этого конкретного пользователя (помним, что у Марии всего 1 карта, а у Ивана — 2!). 
            Если всё совпало — верните название карты.
    План Б (Фоллбек): 
            Если любой из индексов не существует — сабскрипт не должен ронять приложение. Верните строку-заглушку: "💳 Карта не найдена".
            
🧪 Шаблон для ваших тестов:

let bankDB = BankDatabase()

// Тест 1: Успешно берем вторую карту Алексея (Индекс 0, Карта 1)
print(bankDB[0, 1]) // Должно вывести: "Visa"

// Тест 2: Успешно берем единственную карту Марии (Индекс 1, Карта 0)
print(bankDB[1, 0]) // Должно вывести: "Mastercard"

// Тест 3: ТЕСТ НА ОШИБКУ! Пытаемся взять вторую карту Марии (Индекс 1, Карта 1)
// У Марии (Клиент 1) всего одна карта под индексом 0. 
// Обычный Swift тут упадет, но ваш сабскрипт должен сработать безопасно!
print(bankDB[1, 1]) // Должно вывести: "💳 Карта не найдена"

💡 Маленькая подсказка: 
    Чтобы проверить количество карт конкретного пользователя, сначала нужно динамически достучаться до его кошелька по первому индексу: userCards[userIndex]. 
    И уже у этого получившегося массива спросить .count или вызвать .indices.contains(cardIndex).
    Не торопитесь, спокойно подумайте над этим условием guard.
